Tidal Wrapper is a desktop-first monorepo built for long-term maintainability.

## Workspace apps

- `apps/desktop`: Electron shell (main + preload)
- `apps/core`: Product renderer with TanStack Start
- `apps/marketing`: Public website
- `apps/docs`: This documentation site
